2.使用mosdns充当默认dns配置

1）网络-dns-dns重定向关闭
2）开启mosdns
3）daed-global-选择domian++模式
4）daed-dns进阶配置

ipversion_prefer: 4
upstream {
  mosdns: 'tcp+udp://127.0.0.1:5335'
}
routing {
  request {
    qtype(aaaa) -> reject
    qtype(https) -> reject
    qname(geosite:private) -> asis
    fallback: mosdns
  }
}

3.使用daed默认dns分流

1）daed-global-选择domian模式
2）daed-dns进阶配置

upstream {
  alidns: 'udp://223.5.5.5:53'
  googledns: 'tcp+udp://8.8.8.8:53'
}

routing {
  request {
    qname(geosite:cn) -> alidns
    qname(geosite:category-ads-all) -> reject
    fallback: googledns
  }
}
